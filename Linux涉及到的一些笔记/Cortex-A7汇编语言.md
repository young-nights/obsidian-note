# <span class="wathet"><font size=4>常用汇编指令</font></span>
## <font size=3>一、Cortex-A7相关汇编语言</font>
<font size=2>

### 1. 系统寄存器访问指令

| 指令                                       | 说明                                                        |
| ------------------------------------------ | ----------------------------------------------------------- |
| MRS(Move to Register from System register) | 将系统寄存器的值读取到通用寄存器中(系统寄存器 → 通用寄存器) |
| MSR(Move to System register from Register) | 将通用寄存器的值写入到系统寄存器中(通用寄存器 → 系统寄存器) |

这两条指令主要用于操作程序状态寄存器：
- **CPSR** (Current Program Status Register)：当前程序状态寄存器
- **SPSR** (Saved Program Status Register)：保存的程序状态寄存器




### 2. 加载和存储指令

| 指令      | 功能说明                                              | 数据大小     | 特性详解                                                           |
| :-------- | :---------------------------------------------------- | :----------- | :----------------------------------------------------------------- |
| **LDR**   | 从内存**加载32位字**到寄存器<br>(内存 → 寄存器)       | 32位字       | 基指令，地址需4字节对齐                                            |
| **STR**   | 将寄存器的**32位字**写入内存<br>(寄存器 → 内存)       | 32位字       | 基指令，地址需4字节对齐                                            |
| **LDRB**  | 从内存**加载无符号字节**到寄存器<br>(内存 → 寄存器)   | **8位字节**  | 加载到寄存器低8位，**高24位清零**                                  |
| **STRB**  | 将寄存器的**低8位字节**写入内存<br>(寄存器 → 内存)    | **8位字节**  | 只存储寄存器最低8位                                                |
| **LDRSB** | 从内存**加载有符号字节**到寄存器<br>(内存 → 寄存器)   | **8位字节**  | **符号扩展到32位**（如0x80→0xFFFFFF80）                            |
| **LDRH**  | 从内存**加载无符号半字**到寄存器<br>(内存 → 寄存器)   | **16位半字** | 加载到寄存器低16位，**高16位清零**，地址需2字节对齐                |
| **STRH**  | 将寄存器的**低16位半字**写入内存<br>(寄存器 → 内存)   | **16位半字** | 只存储寄存器最低16位，地址需2字节对齐                              |
| **LDRSH** | 从内存**加载有符号半字**到寄存器<br>(内存 → 寄存器)   | **16位半字** | **符号扩展到32位**（如0x8000→0xFFFF8000），地址需2字节对齐         |
| **LDM**   | 从内存**批量加载**多个寄存器<br>(内存 → 寄存器列表)   | **32位字×N** | 支持IA/IB/DA/DB四种模式，`!`更新基址寄存器，用于函数返回恢复现场   |
| **STM**   | 将多个寄存器**批量存储**到内存<br>(寄存器列表 → 内存) | **32位字×N** | 与LDM对应，用于函数入口保存现场，**中断/调用需4字节对齐**          |
| **PUSH**  | 将寄存器**压入堆栈**<br>(寄存器 → 内存\[SP递减])      | **32位字×N** | **STMDB SP!**&#x7684;别名，满递减栈，SP先减后存，**SP需4字节对齐** |
| **POP**   | 从堆栈**弹出寄存器**<br>(内存\[SP递增] → 寄存器)      | **32位字×N** | **LDMIA SP!**&#x7684;别名，弹栈后SP自增，`POP {PC}`用于函数返回    |
